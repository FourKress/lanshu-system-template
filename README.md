# 模板项目

1.安装依赖

``` bash
npm install
# 或
yarn
```


2.项目跑起来

``` bash
npm run start
```

3.修改并提交

# 提交到本仓库
git add .
git commit -m "message"
git push
```

### 提交日志 Prefix

* feat：新功能（feature）
* fix：修补bug
* docs：文档（documentation）
* style： 格式（不影响代码运行的变动）
* refactor：重构（即不是新增功能，也不是修改bug的代码变动）
* test：增加测试
* chore：构建过程或辅助工具的变动
